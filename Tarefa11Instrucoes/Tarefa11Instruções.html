Teclas de Piano

Você é um desenvolvedor web contratado por uma empresa de educação musical. Este cliente quer que você crie um jogo interativo para ajudar seus alunos iniciantes de piano a estudar. Crie um pianista com eventos DOM em JavaScript!

  1. No início do código, temos atribuições de nomes de variáveis para o array keys e o array notes. Há também uma função que percorre os elementos do array keys e os empurra para o array notes. Após o segundo comentário, crie uma função chamada keyPlay que muda a cor de fundo das teclas quando elas são pressionadas. Certifique-se de usar a propriedade .target na função porque o destino está sendo modificado neste caso.
  2. A seguir, crie uma função chamada keyReturn que retorne a cor de fundo das teclas ao seu padrão com uma string vazia '' quando o mouse é liberado sobre o elemento. Certifique-se de usar a propriedade .target.
  3. Agora que você criou duas funções que alteram a cor dos elementos-chave, você deve atribuí-las como os valores das propriedades do manipulador de eventos. Crie uma função nomeada e deixe o bloco de código em branco por enquanto. Essa função será usada mais tarde para atribuir eventos às teclas, portanto, deve receber um parâmetro — você pode chamá-la de nota.
  4. Dentro da função, crie um manipulador de eventos que executa o keyPlay como um manipulador de eventos quando um evento mousedown é acionado em qualquer nota.
  5. Dentro da função, crie uma segunda propriedade do manipulador de eventos que executa o keyReturn quando um evento mouseup é acionado em qualquer nota.
  6. No início do código, temos atribuições de nomes de variáveis para o array keys e o array vazio notes. Há também uma função percorrendo o array keys e empurrando os elementos keys para o array notes para receber um nome de variável. Agora, você deve criar um loop .forEach que passará os elementos no array notes por meio de sua função de atribuição de evento.
  7. Agora, o programa sabe o que fazer quando cada tecla do piano tiver um evento mousedown ou mouseup disparado nela. Execute seu código e veja como funciona!
  8. Em seguida, existem variáveis que representam os botões de progresso na caixa de música abaixo que permitem aos alunos progredir no piano. O nextOne, nextTwo, nextThree e startOver mudam a letra e as notas musicais da música para ajudar o aluno a tocar junto. No início da música, o único botão que o aluno precisa é nextOne. Por causa disso, as propriedades .hidden dos outros botões recebem o valor true. Agora você deve criar eventos em todos os botões de progresso. Primeiro, crie uma propriedade do manipulador de eventos com um evento click no elemento nextOne.
  9. Para começar a modificar a caixa de música, você deve primeiro alternar os botões de progresso. Usando uma função de manipulador de eventos anônimo, faça as seguintes alterações no botão que aparece depois que nextOne é clicado:
    1. Revele o botão nextTwo alterando a propriedade .hidden para fazer o botão nextTwo aparecer.
    2. Oculte o botão nextOne alterando a propriedade .hidden para ocultar o botão nextOne.
  10. Em seguida, um evento de clique acionado no nextOne deve alterar as notas musicais que guiam o aluno de piano pela música. Adicione as seguintes alterações à função do manipulador de eventos nextOne para que as notas musicais mudem quando o botão for clicado. Altere o conteúdo do elemento com um ID de letra-nota-cinco para D. Altere o conteúdo do elemento com um ID de letra-nota-seis para C.
  11. Crie outra propriedade do manipulador de eventos com um evento click no elemento de botão chamado nextTwo. Em seguida, atribua a propriedade ao valor de uma função de manipulador de eventos anônimo.
    12. Faça as seguintes alterações no botão que aparece quando nextTwo é clicado:
    1. Revele o botão nextThree alterando a propriedade .hidden para fazer o botão nextThree aparecer.
    2. Oculte o botão nextTwo alterando a propriedade .hidden para ocultar o botão nextTwo.
  13. Assim que o aluno chegar a esse ponto da música de feliz aniversário, a letra muda de HAP-PY BIRTH-DAY TO YOU para HAP-PY BIRTH-DAY DEAR FRI-END. Faça as seguintes alterações nas letras na função quando o botão for clicado:
    1. Altere o conteúdo do elemento com um ID de palavra cinco para DEAR.
    2. Altere o conteúdo do elemento com um ID de word-six para FRI-.
  14. Agora você tem a letra HAP-PY BIRTH-DAY DEAR FRI-. Para terminar a linha, você deve adicionar -END à caixa de música sob o piano. O elemento -END é armazenado na variável lastLyric. Adicione uma instrução à função do manipulador de eventos para nextTwo que altere a propriedade de exibição de lastLyric para 'inline-block'.
  15. A click event firing on the second button must also change the music notes to guide the piano student through the song. Add the following changes to the nextTwo event handler function so the musical notes change when the button is clicked:
    1. Change the content of the element with an ID of letter-note-three to G.
    2. Change the content of the element with an ID of letter-note-four to E.
    3. Change the content of the element with an ID of letter-note-five to C.
    4. Change the content of the element with an ID of letter-note-six to B.
  16. Crie uma propriedade do manipulador de eventos com um evento click no elemento nextThree.
  17. Usando uma função de manipulador de eventos anônimo, faça as seguintes alterações no botão que aparece quando nextThree é clicado:
    1. Revele o botão startOver alterando a propriedade .hidden para fazer o botão startOver aparecer.
    2. Oculte o botão nextThree alterando a propriedade .hidden para ocultar o botão nextThree.
  18. Adicione as alterações a seguir à função do manipulador de eventos nextThree para que as letras mudem quando esse botão for clicado.
    1. Altere o conteúdo do elemento com um ID de word-one para HAP-.
    2. Altere o conteúdo do elemento com um ID de word-two para PY.
    3. Altere o conteúdo do elemento com um ID de palavra três para BIRTH.
    4. Altere o conteúdo do elemento com um ID de word-four para DAY
    5. Altere o conteúdo do elemento com um ID de palavra cinco para TO.
    6. Altere o conteúdo do elemento com um ID de palavra seis para VOCÊ!.
  19. Adicione as seguintes alterações à função do manipulador de eventos nextThree para que as notas musicais mudem quando o botão for clicado.
    1. Altere o conteúdo do elemento com um ID de letra-nota-um para F.
    2. Altere o conteúdo do elemento com um ID de letra-nota-dois para F.
    3. Altere o conteúdo do elemento com um ID de letra-nota-três para E.
    4. Altere o conteúdo do elemento com um ID de letra-nota-quatro para C.
    5. Altere o conteúdo do elemento com um ID de letra-nota-cinco para D.
    6. Altere o conteúdo do elemento com um ID de letra-nota-seis para C.
  20. Agora você tem a letra FELIZ ANIVERSÁRIO PARA VOCÊ! -FIM — isso não pode estar certo! Para terminar a linha, você deve se livrar do “-end” na caixa de música. Adicione uma instrução à função do manipulador de eventos para nextThree que altere a propriedade de exibição de lastLyric de volta para 'none'.
  21. Parabéns, você completou o Piano Player! Brinque com o piano e a caixa de música para disparar todos os eventos.
